Shell Sort Algorithm

1. Choose a gap sequence (commonly using diminishing powers of 2).
2. Start with the largest gap and perform an insertion sort on elements that are gap distance apart.
3. Reduce the gap and repeat step 2 until the gap is 1.
4. Finally, perform a regular insertion sort.

Time Complexity

Best Case : O(n log n)
Average Case: O(n log n)
Worst Case: O((n log n)^2)
